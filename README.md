## Features

- **Animated Hero** - Particle canvas background with smooth animations
- **Portfolio Showcase** - Interactive project cards with modal details
- **Tech Stack Display** - Animated tech tags with gradient color flow
- **Responsive Design** - Mobile-first, works on all devices
- **Contact Form** - Formspree integration for email delivery
- **Calendly Integration** - Easy booking for consultations
- **Smooth Scrolling** - Section-based navigation
- **SEO Optimized** - Meta tags, Open Graph, Twitter Cards

```bash

npm run dev

# Build for production
npm run build
```

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import to [Vercel](https://vercel.com)
3. Auto-detects Vite settings
4. Deploy

### Netlify

1. Push to GitHub
2. Import to [Netlify](https://netlify.com)
3. Build command: `npm run build`
4. Publish directory: `dist`
